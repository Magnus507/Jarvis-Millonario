# 🚀 Jarvis Millonario

Este es el repositorio central de **Jarvis Millonario**, un espacio colaborativo diseñado para la gestión de proyectos, ideas y conocimiento compartido.

## 🧠 Estructura del Proyecto

El corazón de este repositorio es la **Memoria de Obsidian**, ubicada en:
`Proyectos/Ideas/`

### Carpetas Principales:
- **01_Bandeja_de_Entrada**: Captura rápida de ideas.
- **02_Proyectos**: Gestión activa de proyectos (incluyendo el **Proyecto de Medicina**).
- **03_Recursos**: Documentación y referencias.
- **04_Plantillas**: Formatos estandarizados para notas.

## 🤝 Colaboración

Para trabajar en este proyecto:
1. Asegúrate de tener **Obsidian** instalado.
2. Clona este repositorio.
3. Abre la carpeta `Proyectos/Ideas` como tu Vault principal.
4. Antes de empezar a escribir, haz un `git pull`. Al terminar, haz `git push`.
