# 🏍️ Cilindraje 507: El Juego Nacional Motero

## Visión
Una plataforma gamificada que conecta a los moteros de Panamá con el turismo y el comercio local a través de desafíos físicos y digitales.

## 🕹️ Mecánica del Juego
1. **Registro:** Moteros verificados con foto de su máquina.
2. **Exploración:** Mapa interactivo con ubicación de stickers QR.
3. **Validación:** Escaneo de QR físico + validación GPS (anti-trampa).
4. **Temporadas:** Ciclos de 6 meses con rankings y recompensas.
5. **Evento Final:** "Rodada Nacional" (9 Provincias / 9 Stickers Legendarios).

## 📊 Modelo de Negocio
- **Inscripciones:** Fee por temporada y por evento final.
- **E-commerce:** Venta de stickers oficiales (Normal, Premium, Legendario).
- **Partners:** Suscripción para tiendas/talleres (puntos de escaneo oficiales).
- **Sponsors:** Marcas de motos, lubricantes y equipamiento.

## 🛠️ Stack Técnico (Prioridad Website)
- **Framework:** React + Vite (Velocidad extrema).
- **Estética:** CSS Vanilla (Glassmorphism, Neon Biker Style).
- **Backend:** Supabase (Auth, Real-time Leaderboard, Storage).
- **Geolocalización:** Mapbox API / Leaflet.
