# 🎨 Concepto de Interfaz de Usuario (UI/UX)
---
Proyecto: [[Proyecto de Medicina]]
---

Para este proyecto, la interfaz debe transmitir **confianza, limpieza y eficiencia**. Utilizaremos una estética moderna con los siguientes elementos:

## 1. Paleta de Colores y Estilo
*   **Color Primario:** Azul Médico (Moderno, no aburrido - ej. `#0F52BA` o `#2E5BFF`).
*   **Acentos:** Verde Esmeralda Suave (para salud/éxito) y Blanco Níveo para fondos.
*   **Estilo:** Glassmorphism (efectos de cristal) en las tarjetas de historial para dar profundidad.
*   **Tipografía:** Inter o Roboto (fuentes sans-serif limpias y fáciles de leer).

## 2. Pantallas Principales

### A. Dashboard del Paciente (Timeline)
*   **Visualización:** Una línea de tiempo vertical con iconos que diferencian el tipo de visita (Cardiología, General, Laboratorio).
*   **Interacción:** Al tocar un evento, se expande una tarjeta con el diagnóstico y la receta.
*   **Botón Flotante (QR):** Un botón prominente para generar el código QR de acceso instantáneo para el doctor.

### B. Buscador de Especialistas
*   **Filtros Rápidos:** Botones con iconos para especialidades comunes (Urología, Pediatría, Ginecología).
*   **Resultados:** Tarjetas tipo "Perfil Profesional" con foto, estrellas de rating, y el botón de "Agendar Cita".
*   **Mapa:** Integración con mapas para ver los doctores más cercanos en la provincia seleccionada.

### C. Portal del Doctor
*   **Vista de Consulta:** Pantalla dividida. A la izquierda el historial que el paciente compartió (Timeline), a la derecha un formulario rápido para escribir el nuevo diagnóstico.
*   **Lector de QR:** Acceso rápido a la cámara para escanear el perfil del paciente.

## 3. Micro-interacciones
*   **Efectos Hover:** Las tarjetas de los doctores se elevan ligeramente y brillan al pasar el mouse.
*   **Transiciones:** Deslizamientos suaves (smooth transitions) entre la búsqueda y el perfil del doctor.
